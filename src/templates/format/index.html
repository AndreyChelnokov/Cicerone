Индивидуальные занятия

Для практичных людей, которые ценят свое время, нацелены на результат и хотят за минимальный промежуток времени получить максимум знаний

  • Обучение один на один с преподавателем
  • 2 занятия по 90 минут в неделю
  • Бесплатное пробное занятие
  • График занятий по взаимной договоренности
Групповые занятия

Для любителей общения и людей, которые хотят изучить иностранный язык и при этом немного сэкономить.

  • Группы до 5 человек
  • 2 занятия по 90 минут в неделю
  • Бесплатное пробное занятие
  • Занятия по утвержденному графику
